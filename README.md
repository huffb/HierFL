# HierFL
This Flask-based web application implements a visual interactive system for Hierarchical Federated Learning (HierFL), integrated with image recognition functionality for MNIST/FMNIST datasets. It supports core features including federated learning model training (with real-time progress monitoring), pre-trained model download, handwritten digit/fashion item recognition (with result visualization), and export of recognition results in JSON format. The system adopts asynchronous training via multi-threading, front-end and back-end data interaction through JSON, and temporary file management for visualization assets, serving as a demonstration platform for federated learning algorithm validation and teaching.
